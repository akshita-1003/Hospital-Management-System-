# Hospital Management System

A simple Hospital Management System built using Python and Object-Oriented Programming (OOP) concepts.

## Features

- Add new patients
- Search patients
- Remove patients
- View all patients
- Priority-based patient queue
  - Normal
  - Urgent
  - Super Urgent
- Book doctor appointments
- View appointments
- Generate hospital reports
- Store patient records in a text file

## Technologies Used

- Python
- Object-Oriented Programming (OOP)
- File Handling


## Menu Options

1. Add Patient
2. Show Patients
3. Search Patient
4. Remove Patient
5. Get Next Patient
6. Book Appointment
7. Show Appointments
8. Hospital Report
9. Exit
